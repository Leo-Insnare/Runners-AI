# 2차 라벨링 툴 v0.4.5 검수 시나리오

본 문서는 고객 검수자가 2차 라벨링 툴의 핵심 기능을 순서대로 확인하기 위한 체크리스트입니다.  
문서 패치 v0.4.6에서 현재 앱 탭 구조와 실행 방법 기준으로 정리했습니다.

---

## 1. 실행 확인

### Windows 권장 방식

1. 압축 파일을 해제합니다.
2. 터미널에서 프로젝트 폴더로 이동합니다.
3. 아래 파일을 순서대로 실행합니다.

```bat
setup_windows.bat
run_windows.bat
```

4. 브라우저에서 Streamlit 화면이 열리는지 확인합니다.

### 직접 실행 방식

```bat
rmdir /s /q .venv
py -3.11 -m venv .venv
.venv\Scripts\python.exe -m pip install --upgrade pip setuptools wheel
.venv\Scripts\python.exe -m pip install --no-cache-dir -r requirements.txt
.venv\Scripts\python.exe -m streamlit run app.py
```

Python 3.11이 없다면 `py -3.12`로 바꿔 실행합니다.

---

## 2. 샘플 세션 확인

1. 좌측 사이드바에서 `샘플 세션 추가`를 클릭합니다.
2. `SAMPLE_001` 세션이 생성되는지 확인합니다.
3. 기존 세션 목록에서 `SAMPLE_001`을 선택 후 불러옵니다.
4. 입력값과 샘플 Export 구조를 확인합니다.

샘플 세션은 사용법 확인용이며, 실제 검수/모델링에는 고객 측 실제 영상과 MotionMetrix 값을 사용합니다.

---

## 3. 신규 세션 생성 및 세션 정보 저장

1. 좌측 사이드바에서 `새 세션 생성`을 클릭합니다.
2. 현재 세션 ID가 생성되는지 확인합니다.
3. `1. 세션 정보` 탭에서 아래 값을 입력합니다.
   - 사용자 ID
   - 나이
   - 성별
   - 키
   - 체중
   - 달리기 속도
   - 측면/후면 FPS
   - 측정 시작/종료 시간
4. `세션 정보 저장`을 클릭합니다.
5. 저장 완료 메시지가 표시되는지 확인합니다.

---

## 4. 촬영 Wizard / Overlay 확인

1. `2. 촬영 Wizard/Overlay` 탭으로 이동합니다.
2. 작업 순서를 선택합니다.
   - 후면 전신 정지
   - 후면 달리기
   - 측면 전신 정지
   - 측면 달리기
3. 확인할 계산 항목을 선택합니다.
4. 저장된 영상 프레임 Preview 또는 카메라 스냅샷 방식으로 Skeleton Overlay를 생성합니다.
5. 아래 항목이 표시되는지 확인합니다.
   - Skeleton Point
   - 선택 지표에 필요한 포인트 강조
   - 수직선/지면선/몸통선 등 기준선
   - 현재 프레임 기준 참고 계산값
6. Preview 이미지를 저장합니다.

검수 기준은 **저장된 영상 프레임 Preview**입니다. Live Camera Stream은 실험 기능이므로 필수 검수 항목으로 보지 않습니다.

---

## 5. 영상 업로드 확인

1. `3. 영상 업로드` 탭으로 이동합니다.
2. 아래 영상을 각각 업로드합니다.
   - 측면 정지 영상
   - 측면 달리기 영상
   - 후면 정지 영상
   - 후면 달리기 영상
3. 저장된 영상 목록에 파일명과 용량이 표시되는지 확인합니다.
4. 저장 후 `videos.json`과 Export CSV에 영상 경로가 포함되는지 확인합니다.

---

## 6. 측면 MotionMetrix 수치 입력 확인

1. `4. MotionMetrix 입력 - 측면` 탭으로 이동합니다.
2. 아래 주요 항목을 입력합니다.
   - Forward Lean
   - Overstride
   - Braking Force
   - Hip/Thigh Flexion Extension
   - Knee Flexion / Knee ROM
   - Shank Angle
   - Foot Strike Type
   - Cadence
   - Contact Time
   - Vertical Oscillation
   - Step / Stride Length
   - Flight Time
3. 각 지표의 `사용 Skeleton Point / 계산 기준` 안내가 표시되는지 확인합니다.
4. 사이드바의 `필수값만 보기`를 켜고 core 필수 항목만 표시되는지 확인합니다.
5. 일반적인 범위를 벗어난 값을 입력했을 때 경고가 표시되는지 확인합니다.

---

## 7. 후면 지표 입력 확인

1. `5. MotionMetrix 입력 - 후면` 탭으로 이동합니다.
2. 아래 항목을 입력합니다.
   - Pelvic Drop
   - Knee Medial Collapse
   - Step Width / Crossover
   - Trunk Lateral Tilt
3. 후면 지표별 Skeleton Point와 계산 기준이 표시되는지 확인합니다.
4. 입력 후 저장합니다.

---

## 8. 종합/선택 입력 확인

1. `6. 종합/선택 입력` 탭으로 이동합니다.
2. 아래 항목을 입력합니다.
   - Running Economy
   - Running Type
   - Vertical Force optional
   - Lateral Force optional
   - Stride Rating optional
3. 단위 선택값이 있는 경우 결과지 기준 단위를 선택합니다.
4. 입력 후 저장합니다.

---

## 9. 육안 자세 평가 확인

1. `7. 육안 평가` 탭으로 이동합니다.
2. 아래 항목에 대해 강도/방향/메모를 입력합니다.
   - 걷듯이 뛴다
   - 뒤뚱거린다
   - 좌우 비대칭
   - 팔 동작 비대칭
   - 상체 좌우 흔들림
   - 무릎 상승 부족
3. 화면에는 한글 선택값이 보이고 저장값은 모델링용 영문 코드로 저장되는지 확인합니다.

---

## 10. Skeleton Guide 확인

1. `8. Skeleton Guide` 탭으로 이동합니다.
2. 전체 Skeleton Point registry가 표시되는지 확인합니다.
3. 지표별 required points, derived points, calculation basis가 표시되는지 확인합니다.

---

## 11. 검토 / Export 확인

1. `9. 검토/Export` 탭으로 이동합니다.
2. 필수 입력 완료율과 누락 항목을 확인합니다.
3. `전체 Export 생성`을 클릭합니다.
4. 아래 파일이 생성되는지 확인합니다.

```text
training_dataset_wide.csv
training_dataset_long.csv
metric_dictionary.csv
missing_value_report.csv
data_quality_report.csv
data_quality_summary.csv
```

---

## 12. 저장/불러오기 확인

1. 좌측 사이드바에서 `현재 세션 저장`을 클릭합니다.
2. 기존 세션 목록에서 방금 저장한 세션을 선택합니다.
3. `선택 세션 불러오기`를 클릭합니다.
4. 입력값, 영상 경로, Preview 요약, 육안 평가값이 복원되는지 확인합니다.

---

## 13. 백업 확인

1. `전체 백업 ZIP 생성`을 클릭합니다.
2. 생성된 ZIP 파일을 다운로드합니다.
3. 세션 데이터, metric definitions, export 파일이 포함되어 있는지 확인합니다.

---

## 14. 합격 기준

아래 항목이 정상 동작하면 2차 라벨링 툴 검수 기준을 충족한 것으로 봅니다.

- 세션 생성/저장/불러오기 가능
- 측면/후면 영상 업로드 및 경로 저장 가능
- 촬영 Wizard/Overlay Preview 확인 가능
- MotionMetrix 결과값 직접 입력 가능
- Skeleton Point와 계산 기준 확인 가능
- 육안 자세 평가 가능
- 필수값 누락 확인 가능
- CSV/JSON Export 가능
- 데이터 품질 리포트 생성 가능
- 전체 백업 ZIP 생성 가능
