# 2차 라벨링 툴 v0.4.5 범위 안내

본 Streamlit 앱은 2차 프로젝트 범위인 라벨링 툴입니다. 고객 워드 문서의 작업 흐름을 반영하여 촬영 순서 Wizard와 Skeleton Overlay Preview를 포함합니다.

---

## 1. 포함 범위

- MotionMetrix 결과값 직접 입력
- 측면/후면 영상 세션 관리
- 사용자 정보, 속도, FPS, 측정 구간 입력
- 고객 문서 순서 기반 촬영 Wizard
  - 후면 전신 정지 2~3초
  - 후면 달리기 촬영/확인
  - 측면 전신 정지 2~3초
  - 측면 달리기 촬영/확인
- 업로드 영상 프레임 기반 Skeleton Overlay Preview
- 카메라 스냅샷 기반 Skeleton Overlay Preview
- 선택 지표별 필요한 Skeleton Point 강조 표시
- 수직선, 지면선, 몸통선 등 계산 기준선 표시
- 현재 프레임 기준 참고 계산정보 표시
- Skeleton Point 및 계산 기준 확인
- 육안 자세 평가 입력
- 세션 단위 JSON 저장
- 3차 모델링용 CSV Export
- 누락값 리포트 및 데이터 품질 리포트
- Preview 이미지 및 요약정보 Export
- 전체 백업 ZIP 생성

---

## 2. 실험/선택 기능

- Live Camera Stream 기반 Skeleton Overlay

이 기능은 브라우저 권한, Streamlit Cloud 환경, 네트워크 상태, WebRTC 동작 여부에 따라 차이가 있을 수 있습니다. 고객 검수 안정성은 저장된 영상 프레임 Preview 방식을 우선 권장합니다.

---

## 3. 미포함 범위

아래 기능은 3차 프로젝트 또는 별도 협의 범위입니다.

- MotionMetrix 수준의 자동 수치 산출 보장
- Braking Force 최종 자동 산출
- Running Economy 최종 자동 산출
- Running Type 최종 자동 분류
- STT/LLM 기반 음성 자동 입력
- 위험도/교정 처방 추론
- REST API 서버
- MotionMetrix 파일 자동 파싱
- 모바일 앱 내 카메라 촬영 UI
- 운영 서버 구축 또는 상용 서비스 수준의 장애 대응
- 실시간 분석 성능 보장

---

## 4. 검수 시 유의사항

- Overlay 계산값은 라벨링 보조용 참고값입니다.
- 최종 학습 정답값은 MotionMetrix 수치를 직접 입력한 값을 기준으로 저장합니다.
- 단일 프레임에서 계산할 수 없는 지표는 포인트 확인과 기준선 확인 중심으로 표시됩니다.
- Contact Time, Cadence, Flight Time, Vertical Oscillation, Braking Force, Running Economy 등은 3차에서 구간/이벤트 기반 계산 또는 보정 모델로 연결됩니다.
- Streamlit Cloud는 테스트용 경량 환경이므로 Export와 백업 ZIP을 자주 다운로드해야 합니다.
- 고객 측 MotionMetrix 결과지 양식과 단위에 따라 필드명/입력 순서는 일부 조정될 수 있습니다.

---

## 5. 고객에게 설명할 핵심 문구

```text
본 툴은 2차 라벨링 툴 개발 산출물입니다.
저장된 측면/후면 촬영 영상을 업로드한 뒤 Skeleton Overlay Preview와 지표별 계산 기준을 확인하고,
MotionMetrix 결과값을 직접 입력하여 3차 모델링용 학습 데이터를 생성합니다.

Preview/Live 값은 라벨링 보조용 참고값이며,
최종 학습 정답값은 MotionMetrix 직접 입력값입니다.
```
